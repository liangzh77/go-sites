1. 游戏概述

游戏简介

狼人杀（Werewolf/Mafia）是一种社交推理桌游，玩家分为好人阵营（村民）和坏人阵营（狼人）。好人需通过发言、推理和投票找出并处决狼人；狼人则需夜晚杀戮并伪装身份，直到人数优势。

  • 核心机制：夜晚闭眼行动 + 白天公开讨论。
  • 玩家人数：5-18人（新手推荐8-12人，小局5-7人需简化规则）。
  • 一局时长：20-60分钟。
  • 游戏益处：提升逻辑思维、观察力和社交技能，充满惊喜与欢笑。

快速开始

  1. 准备身份牌（总牌数=玩家数）。
  2. 主持人（经验玩家）主持流程。
  3. 抽牌保密，游戏开始！
互动提示：分享一句游戏台词，如"我是预言家，你是狼！" 以激发兴趣。

2. 基础操作：规则、角色与术语

本节聚焦核心玩法。新手先掌握此部分，建议模拟一局闭眼环节。

基础规则

游戏循环：夜晚行动 → 白天讨论 → 投票。主持人控制指令。

3. 基础实战：入门策略

规则上手后，焦点转向"怎么赢"。新手原则：听多说少，逻辑优先。

核心策略

  1. 观察发言：注意语气、细节。狼人常回避昨晚事件或逻辑矛盾。
  2. 投票决策：优先票高疑玩家（e.g., 自称神职无证据者）。跟风或质疑。
  3. 心态管理：第一局多听，被票时冷静辩护。
  4. 避坑指南：
    • 别信假预言家（问："昨晚查谁？证据？"）。
    • 别乱自爆身份。

实战案例

场景

有人自称预言家，说"5号金水"。

策略

追问细节，若支吾 → 票他。结果：村民胜。

互动提示：脑storm：你是平民，有人承认狼人，你怎么票？（答：确认真假后票。）

4. 进阶扩展：深化规则与角色

玩2-3局基础后扩展。保持牌平衡（神:狼:民 ≈ 3:3:4）。

进阶规则

  • 多狼模式：狼人3-4人，夜晚协作刀。
  • 复活/守护：加守卫角色，夜晚守护1人免刀。

5. 精通层面：高手策略

多玩10局后，进入心理战。记录每局心得。高手策略强调多维度博弈：逻辑、心理、团队。以下详细描述核心技巧，结合实战应用。

高手技巧

1. 心理博弈：读微表情与制造假象

高手通过观察非语言线索（如眼神回避、犹豫停顿）判断身份，准确率可达70%以上。狼人常利用此制造假象，例如悍跳预言家时，先自刀不上警（避免暴露），再用"怂"发言（如承认小错）提升身份安全感。好人则模拟神职发言，误导狼队：例如平民假装预言家，报假金水拉票。实战中，狼人可对女巫毒药威胁下"穿衣"（刀女巫），迫使好人内斗。Tips：多练读人，结合逻辑（如发言连贯性）验证。

2. 团队协作：默契沟通与统一票型

狼队夜晚"脑内会议"选刀目标，优先刀强神（如预言家）；白天用暗号（如特定短语）协调。村民间通过信息共享（如"昨晚刀口逻辑"）统一视角，避免内斗。高手强调"枪徽流"：用猎人枪或警徽拉票，制造威胁。案例：在4狼局，狼队一狼悍跳拿警徽，另一狼冲锋站边，成功扛出真预言家。Tips：村民用"保身份"对话（如逻辑链条）协同；狼队防"倒钩"（队友被票时反拉）。

3. 全局分析：反推逻辑与算牌

高手从"若A是狼，则B发言不对"反推全局，结合剩余牌算狼坑（e.g., "神全跳，死2人，剩3狼"）。狼人策略：首夜刀边缘玩家，避开核心神；好人用"打煽动"强势拉票（给弱好人逻辑保底）。数学模型：博弈论视角，预言家查验优先高威胁位，胜率提升30%。Tips：记录发言表格，复盘"信息不对称"点。

4. 避坑与进阶应用：忽略安静玩家与过度自信

常见坑：安静玩家（警下）常藏狼，高手必追问。防过度自信：狼人别早穿女巫（毒药克制）；好人别信全金水。应用：用"诈身份"测试（如发后置位查杀），或在公投前"裸冲"拉票。长期：观摩高手局，模拟AI对决提升。
