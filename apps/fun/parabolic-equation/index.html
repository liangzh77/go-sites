抛物线知识

参数 a 的影响

系数 a 控制抛物线的开口方向和宽窄程度：

  • 当 a > 0 时，抛物线开口向上
  • 当 a < 0 时，抛物线开口向下
  • |a| 值越大，抛物线开口越窄；|a| 值越小，抛物线开口越宽

顶点计算

抛物线 y = ax² + bx + c 的顶点坐标为：

x = -b/(2a), y = c - b²/(4a)

顶点是抛物线的最高点或最低点，也是抛物线的对称轴与抛物线的交点。

根的性质

判别式 Δ = b² - 4ac 决定了抛物线与 x 轴交点（根）的数量：

  • 当 Δ > 0 时，抛物线与 x 轴有两个不同的交点（两个不相等的实根）
  • 当 Δ = 0 时，抛物线与 x 轴有一个交点（两个相等的实根）
  • 当 Δ < 0 时，抛物线与 x 轴没有交点（没有实根，有两个共轭复根）
