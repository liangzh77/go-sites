多项式求导公式

从定义出发，推导幂函数的导数公式

📐 基础定义

f'(x) = limh→0 [f(x+h) - f(x)] / h

导数的定义：函数在某一点的导数等于该点处切线的斜率，通过极限的方式求得。

🔬 推导过程

步骤 1: 设定幂函数

设 f(x) = xn，其中 n 为正整数

步骤 2: 应用导数定义

f'(x) = limh→0 [(x+h)n - xn] / h

步骤 3: 二项式展开

(x+h)n = xn + nxn-1h + Cn2xn-2h2 + ... + hn

根据二项式定理展开

步骤 4: 代入化简

f'(x) = limh→0 [nxn-1h + Cn2xn-2h2 + ... + hn] / h

= limh→0 [nxn-1 + Cn2xn-2h + ... + hn-1]

步骤 5: 取极限

当 h→0 时，所有含 h 的项都趋向于 0

因此：f'(x) = nxn-1

结论：(xn)' = nxn-1

📊 推广到一般多项式

对于多项式 f(x) = anxn + an-1xn-1 + ... + a1x + a0

求导规则：

1. 常数的导数为 0：(c)' = 0

2. 幂函数求导：(xn)' = nxn-1

3. 常数倍：(cf(x))' = cf'(x)

4. 和的导数：(f+g)' = f' + g'

f'(x) = nanxn-1 + (n-1)an-1xn-2 + ... + a1
